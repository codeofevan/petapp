步骤三、项目结构解析 
	d build 构建脚本目录 
	d config 构建配置目录
		！修改打包后文件路径——index.js 文件 
	d node_modules 依赖工具包目录 
	d src 源码目录
	-d assets 资源目录
	-d components 公共组件目录
	-d *page 页面相关组件目录
	-d *fetch 存放所有axios处理和封装
	-d router 路由文件目录
	-f App.vue 入口组件 
	-f main.js 入口页面文件 
	d static 静态文件
	f .babelrc babel 配置 
	f index.html 入口页面 
	f package.json 依赖描述文件
